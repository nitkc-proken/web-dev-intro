Visual Studio Code(VSCode)はMicrosoftが開発したエディタ(コードを書くのが得意なアプリ)です。
似たような名前で「Visual Studio」(紫のロゴ)があるので注意して下さい！

# おすすめ拡張機能

## テーマ

https://marketplace.visualstudio.com/search?target=VSCode&category=Themes&sortBy=Installs

VSCodeにはたくさんのテーマがあり、見た目を自由にカスタマイズできます


## Japanese Language Pack for Visual Studio Code

https://marketplace.visualstudio.com/items?itemName=MS-CEINTL.vscode-language-pack-ja

VSCodeを日本語にしてくれます。
慣れるまでは日本語のほうがいいかもしれません。

## GitHub Copilot

https://marketplace.visualstudio.com/items?itemName=GitHub.copilot

時代はAIです。
学生ならGitHub Copilotの有料版がタダで使えるので、使ってみましょう。


## GitLens

https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens

VSCode上でGitが結構使いやすくなります。フル機能は有料ですが、これも学生ならタダです。
俺はCUIしか認めん！という人には必要ありません。

## Error Lens

https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens

エラーをエディタ上に直接表示してくれます。エラーの内容をいちいち確認しなくてもいいので便利です。

## Pretty TypeScript Errors

https://marketplace.visualstudio.com/items?itemName=yoavbls.pretty-ts-errors

TypeScriptの長ったらしいエラーを見やすくしてくれます。
ないよりはマシです。というか入れたほうがいい。
